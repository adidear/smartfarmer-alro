# 🌾 ระบบฐานข้อมูลเกษตรกรปราดเปรื่อง (Smart Farmer)
### สำนักพัฒนาและถ่ายทอดเทคโนโลยี (สพท.) — สำนักงานการปฏิรูปที่ดินเพื่อเกษตรกรรม (ส.ป.ก.)

---

## 📌 ภาพรวมโครงการ

ระบบฐานข้อมูลเกษตรกรปราดเปรื่อง (Smart Farmer) พัฒนาด้วย **Google Sheets + Google Apps Script** เพื่อใช้เป็นต้นแบบการจัดเก็บ ติดตาม และประเมินผลเกษตรกรในเขตปฏิรูปที่ดิน ส.ป.ก. รองรับเจ้าหน้าที่หลายจังหวัดพร้อมกันผ่าน Web Application URL เดียว โดยไม่ต้องติดตั้งโปรแกรมเพิ่มเติม

> ระบบนี้พัฒนาขึ้นเป็นผลงานตำแหน่งชำนาญการ ระดับ สพท. โดยใช้ Claude AI (claude.ai) เป็นเครื่องมือช่วยออกแบบโครงสร้างและเขียนโค้ด Apps Script ตลอดกระบวนการพัฒนา

---

## 🗂 โครงสร้างฐานข้อมูล 3 ตาราง

```
Master Table          Plot Table              History Table
(ข้อมูลบุคคล)         (ข้อมูลแปลงที่ดิน)       (ประวัติรายปี)
─────────────         ───────────────          ──────────────
เลขบัตร (PK) ◄───── เลขบัตร (FK)             เลขบัตร (FK) ─────►Master
ชื่อ-นามสกุล          แปลงเลขที่               ปีงบประมาณ
จังหวัด               กลุ่ม/ระวาง ส.ป.ก.       กิจกรรมเกษตร
การศึกษา             พื้นที่ (ไร่-งาน-วา)       รายได้ก่อน/หลัง
สมาชิกกลุ่ม           สิ่งอำนวยความสะดวก        ความเปลี่ยนแปลง 7 ด้าน
สถานะ SF สูงสุด      (น้ำ/ไฟ/ถนน/สัญญาณ)       ผลการประเมิน SF
```

**สถานะ Smart Farmer** มี 3 ระดับ เรียงจากต่ำสุดไปสูงสุด:
`Developing` → `Existing` → `ต้นแบบ`

ระบบจะเก็บ **สถานะสูงสุด** ที่เกษตรกรเคยได้รับตลอดทุกปี ไม่ถูกเขียนทับด้วยสถานะที่ต่ำกว่า

## 🖥 Web Application

Web App พัฒนาด้วย Google Apps Script ประกอบด้วย 5 หน้าหลัก:

### 📊 Dashboard
- สถิติรวมแยกตามสถานะ (ต้นแบบ / Existing / Developing)
- กราฟ Doughnut แยกสถานะ
- กราฟ Bar แยกจังหวัด (Stacked)
- เปรียบเทียบรายได้เฉลี่ยก่อน/หลังเข้าร่วมโครงการ
- ตารางสรุปรายปีงบประมาณ
- รายการบันทึกล่าสุด 10 รายการ (คลิกดูโปรไฟล์รายบุคคลได้)

### 👤 ลงทะเบียนเกษตรกรใหม่
- ตรวจสอบเลขบัตรซ้ำแบบ Real-time ก่อนบันทึก
- บันทึกเข้า Master Table พร้อมสูตร COUNTIF/MAXIFS อัตโนมัติ
- รองรับกรณีไม่ได้เป็นเจ้าของแปลง (ระบุความสัมพันธ์กับเจ้าของ)
- Dropdown จังหวัด 77 จังหวัดทั่วประเทศ

### 📋 บันทึกประวัติการพัฒนาประจำปี
- VLOOKUP ดึงชื่อเกษตรกรจาก Master อัตโนมัติเมื่อกรอกเลขบัตร
- คำนวณรายได้เพิ่ม/ลด เปอร์เซ็นต์ อัตโนมัติ
- บันทึกกิจกรรมเกษตร 8 ประเภท ช่องทางการขาย ความรู้ไปใช้ ความเปลี่ยนแปลง 7 ด้าน
- Dropdown จังหวัดที่ตั้งแปลง 72 จังหวัดที่มีเขต ส.ป.ก.
- อัปเดตสถานะใน Master เฉพาะเมื่อสถานะใหม่สูงกว่าเดิม

### 🔍 ค้นหาเกษตรกร
- ค้นหาด้วยเลขบัตรหรือชื่อ-นามสกุล
- กดดูโปรไฟล์รายบุคคลได้ทันที

### 👁 โปรไฟล์รายบุคคล
- ข้อมูลพื้นฐานครบถ้วน
- ข้อมูลแปลงที่ดิน ส.ป.ก. (deduplicate unique แปลง)
- ประวัติการพัฒนาทุกปีพร้อมกราฟรายได้เปรียบเทียบ
- badge สถานะสูงสุด

```
├── Search.html          # Sidebar ค้นหา (สำหรับใช้ใน Sheets)

```

## 📥 การนำเข้าข้อมูลเดิม

รองรับการนำเข้าข้อมูลจากแบบฟอร์มติดตามผล Smart Farmer เดิม 2 วิธี:

### วิธีที่ 1 — ผ่านเมนู ⚡ Smart Farmer ใน Sheets (แนะนำ)
1. อัปโหลดไฟล์ `.xlsx` เข้า Google Drive
2. เปิด Sheets → เมนู **⚡ Smart Farmer → 📥 นำเข้าข้อมูลจาก CSV**
3. ใส่ชื่อไฟล์และปีงบประมาณ → กดนำเข้า
4. ระบบตรวจซ้ำอัตโนมัติ ข้ามเลขบัตรที่มีอยู่แล้ว

### วิธีที่ 2 — Python script (สำหรับไฟล์ที่มีปัญหาพิเศษ)
```bash
pip install pandas openpyxl
python3 sf_converter.py ชื่อไฟล์.xlsx 2568
```
รองรับ: เลขไทย (๑๒๓), เลขมีขีด (3-900200-439-63-1), Scientific Notation (3.62E+12)

---

## ⚙ วิธีติดตั้ง

1. สร้าง Google Sheets ใหม่ ตั้งชื่อ **ฐานข้อมูล Smart Farmer**
2. สร้าง 3 Sheets: `Master`, `Plot`, `History` พร้อมหัวคอลัมน์ตามแผน
3. เปิด **Extensions → Apps Script**
4. วางโค้ดจากไฟล์นี้ตามลำดับ: `Code.gs`, `WebApp.html`, `Import.html`, `Register.html`, `History.html`, `Search.html`
5. เพิ่ม **Spreadsheet ID** ใน `Code.gs`:
   ```javascript
   const SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID';
   ```
6. เพิ่ม **Drive API v2** ใน Services
7. **Deploy → New deployment → Web app**
   - Execute as: Me
   - Who has access: Anyone
8. Reload Sheets จะปรากฏเมนู **⚡ Smart Farmer**

## ⚠ ข้อจำกัด

- Google Apps Script มีเวลาตอบสนอง **3–8 วินาที** ต่อคำขอ เนื่องจากข้อจำกัดของ platform
- ประสิทธิภาพเหมาะสมกับข้อมูลไม่เกิน **5,000 แถว** ต่อ Sheet
- ระบบนี้เป็น **ต้นแบบ** สำหรับนำเสนอผู้บริหาร หากขยายผลระดับประเทศควรพัฒนาสู่ RDBMS
